To find explanations concerning the format of temporal dictionary 
in Xip/Python, refer to the document 
"detailed_description_of_temporal_processing.doc"
in doc directory

The ISO 8601 format, from which TimeML and our representation is
inspired, is explained there:
http://fr.wikipedia.org/wiki/ISO_8601

TimeML format is explained there:
http://timeml.org/site/publications/specs.html

But TIDES guidelines, explaining in much more details the format
of temporal expressions themselves, can be found in file:
MTRAnnotationGuide_v1_02.pdf
in doc directory.
