To run a particular grammar:
   - parameter -grm or property GRM
   - parameter or property XIP_FILES to add some grammar files (option -p in XIP)
   - -printsn to print the number of the parsed sentence 
   - -printst to print the text of the parsed sentence

   - Input options: -text, -xmltext, -dir, -xmldir
   - Input options: -o, odir
