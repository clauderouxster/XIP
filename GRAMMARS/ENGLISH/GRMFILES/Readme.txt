###############################################
# XIP ENGLISH GRAMMAR DIRECTORY FOR GRM FILES #
###############################################


- Each grammar configuration is put under a sub
  directory of the GRMFILES directory:
  Each of these sub-directory contains
  - a grm file describing the path and files
    to use in a given version of the grammar 
  - a configuration files

For example, under ENTITIES_ONLY/, one finds:
    - entities_only.grm : the grm file to be called
    by XIP for this version of the grammar
    - english_entit.xip : the xip grammar configuration 
      proper.
